# AIBOS Security / Corteris Tender AI 1.2.1

## Database Reliability Release

Релиз завершает инфраструктурный этап локальной SQLite-базы.

Реализовано:

- автоматическое обновление старой схемы без удаления данных;
- перевод старых INTEGER ID в UUID с сохранением связей;
- версионирование схемы и журнал миграций;
- резервная копия перед миграцией;
- восстановление исходной базы при неудачной миграции;
- ручные резервные копии с SHA-256 и JSON-метаданными;
- проверка резервной копии перед восстановлением;
- ротация старых копий;
- диагностика SQLite, WAL, foreign keys, таблиц, индексов и количества записей;
- VACUUM, ANALYZE, PRAGMA optimize и checkpoint WAL;
- экспорт базы;
- раздел «Диагностика БД» в настройках;
- исправлена работа UI с UUID тендеров.

Перед восстановлением базы приложение создаёт страховочную копию. После восстановления требуется перезапуск приложения.
