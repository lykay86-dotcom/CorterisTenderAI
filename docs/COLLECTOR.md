# Corteris Tender Collector

C1–C20 не заменяют RM-001–RM-200.

Каждый адаптер обязан иметь отдельные: авторизацию, транспорт, нормализацию, provenance, freshness, timezone, диагностику и тест подключения.

Поддерживаемые варианты: REST, GraphQL, SOAP, RSS/Atom, JSON/XML feed, CSV/XLSX, HTML, browser-assisted, file import, webhook.

Пользовательский конструктор адаптеров не выполняет произвольный Python-код.
