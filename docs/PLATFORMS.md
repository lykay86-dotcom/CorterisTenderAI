# Перечень площадок для Corteris Tender AI

## Основной обязательный контур
1. ЕИС / zakupki.gov.ru — поиск и документы по 44-ФЗ и 223-ФЗ.
2. Сбербанк-АСТ.
3. Росэлторг / ЕЭТП.
4. РТС-тендер.
5. Национальная электронная площадка.
6. ЗаказРФ / АГЗ РТ.
7. Российский аукционный дом / РАД.
8. ЭТП ГПБ.
9. ТЭК-Торг.
10. АСТ ГОЗ — только при наличии законных оснований, допуска и соответствующих требований.

## Коммерческие и корпоративные
- B2B-Center.
- Фабрикант.
- ЭТП ГПБ: корпоративные секции.
- ТЭК-Торг: корпоративные секции.
- Сбер А.
- РТС-маркет и региональные магазины малых закупок.
- Портал поставщиков Москвы.
- Корпоративные закупочные порталы крупных заказчиков.

## Порядок реализации коннекторов
1. Импорт номера/ссылки из ЕИС.
2. Импорт скачанного архива.
3. Официальные открытые данные/API ЕИС.
4. Электронная почта и RSS.
5. Официальные API агрегаторов.
6. Личные кабинеты площадок — только если правила площадки разрешают автоматизацию.

Запрещено обходить CAPTCHA, 2FA, платный доступ и технические ограничения.
