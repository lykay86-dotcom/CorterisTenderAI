# Руководство пользователя
1. Запустите Corteris Tender AI.
2. На вкладке «Добавить» создайте карточку.
3. Выберите тендер в таблице.
4. На вкладке «Анализ» импортируйте ZIP/PDF/DOCX/XLSX.
5. Нажмите «Анализировать».
6. Проверьте цитаты и риски вручную.
7. Нажмите «Сформировать пакет».

Сформированные документы находятся в `%LOCALAPPDATA%\Corteris\CorterisTenderAI\outputs`. Программа не подаёт заявку и не использует ЭП.
