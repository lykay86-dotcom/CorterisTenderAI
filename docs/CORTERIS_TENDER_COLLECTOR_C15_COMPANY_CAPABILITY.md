# Collector C15 — Company Capability Profile

C15 заменяет production-предположения рейтинга редактируемым подтверждённым
профилем ООО «КОРТЕРИС».

## Хранение

Профиль сохраняется в пользовательском каталоге данных:

```text
company_capability_profile.json
```

Запись выполняется атомарно через временный файл. Денежные значения хранятся
десятичными строками. Старый или повреждённый файл не мешает запуску: система
возвращает пустой неподтверждённый профиль.

## Подтверждение

Профиль нельзя сохранить без названия компании, имени подтверждающего и
явного флага подтверждения. Пустые разделы не заполняются встроенными
догадками и отображаются как недостаток данных.

## Данные

Модель поддерживает направления, регионы самостоятельной работы и партнёров,
лицензии и виды работ, СРО, квалификации, бригады, контракты и опыт,
финансовые лимиты, обеспечения, банковскую гарантию, оборудование, бренды,
поставщиков, склад, маржинальность, оплату, отсрочку, собственные работы,
субподряд, нежелательные объекты, проектировщиков и региональных партнёров.

## Рейтинг

Автоматический Collector загружает профиль перед каждым новым запуском.
Ручной пересчёт в реестре загружает профиль перед каждой оценкой. Если профиль
не подтверждён, компоненты регионов, финансов, оборудования, опыта и лицензий
показывают «Недостаточно данных», а итог не может быть выше зоны ручной
проверки.

Встроенный `DEFAULT_CORTERIS_COMPANY_PROFILE` сохранён только для обратной
совместимости старых API и тестовых сценариев. Production composition roots
его больше не используют.
