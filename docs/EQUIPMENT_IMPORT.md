# Импорт каталога оборудования

Поддерживаются CSV и XLSX. Первая строка должна содержать названия столбцов.

Распознаваемые поля:

- Категория
- Бренд
- Модель
- Артикул
- Поставщик
- Закупочная цена
- Розничная цена
- Остаток
- Срок поставки
- Гарантия
- Страна

Обязательные поля: `Бренд` и `Модель`. Повторная строка с тем же брендом, моделью и артикулом обновляет карточку.
