# Collector C14 — интерфейс достоверности и разрешение конфликтов

## Назначение

C14 отображает происхождение критичных полей в реестре и позволяет
пользователю явно выбрать значение при расхождении источников.

## Интерфейс

В реестре добавлены:

- колонка «Достоверность»;
- текстовый статус, а не только цвет;
- всплывающая сводка подтверждённых полей и конфликтов;
- кнопка «Достоверность и источники».

В карточке достоверности показываются:

- все критичные поля;
- выбранное системой значение;
- все кандидаты последней проверки;
- источник и URL;
- уровень доверия;
- официальный/проверенный статус;
- confidence;
- время получения;
- конфликт;
- ручной выбор;
- журнал решений.

## Ручное решение

Ручной выбор:

1. не удаляет альтернативные значения;
2. сохраняется в отдельной таблице;
3. применяется к основной карточке тендера;
4. обновляет эффективный статус конфликта;
5. записывается в неизменяемый журнал;
6. переносится в следующую проверку как пользовательское решение;
7. может быть снят, после чего восстанавливается автоматический приоритет.

## Таблицы схемы v4

```text
collector_tender_field_manual_selections
collector_tender_field_resolution_history
```

## Важное ограничение

Ручной выбор не превращает неофициальный источник в официальный. Уровень
доверия и происхождение поля остаются исходными. Интерфейс показывает, что
значение выбрано вручную.
