# Collector C7 — интерфейс источников

## Назначение

C7 добавляет в существующий `TenderSearchUiController` окно управления
провайдерами. Открытие окна не выполняет сетевых запросов.

## Данные

Нечувствительные переключатели:

```text
<data>/collector_provider_settings.json
```

Результаты проверок:

```text
<data>/collector_provider_health.json
```

Настройки коммерческих площадок без секретов:

```text
<data>/commercial_provider_settings.json
```

API-ключи и bearer-токены в эти файлы не записываются.

## Цвета

- зелёный — источник ответил успешно;
- жёлтый — ограниченный режим, не настроен или требуется ручная проверка;
- красный — ошибка подключения;
- серый — отключён пользователем;
- синий — состояние ещё не определено.

## Сетевое поведение

Проверка выполняется только после нажатия кнопки. Все проверки работают
в `QThreadPool`; GUI не блокируется. На одну операцию создаётся отдельный
`CollectorNetworkRuntime`, который закрывается в `finally`.

## Коммерческие площадки

Коммерческие адаптеры отображаются в интерфейсе, но не считаются рабочими.
Включение означает только желание использовать источник. Пока не подтверждены
договор, ключ, endpoint и реальный API-контракт, проверка возвращает честное
состояние `not_configured` или `unknown`.
