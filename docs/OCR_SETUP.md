# Настройка OCR

1. Установите Tesseract OCR для Windows.
2. При установке выберите языки Russian и English.
3. Добавьте каталог Tesseract в PATH.
4. В командной строке проверьте: `tesseract --version`.
5. Программа автоматически использует языки `rus+eng` для JPG, PNG и TIFF.

Если Tesseract отсутствует, приложение не завершается с ошибкой: документ получает пометку «OCR недоступен».
