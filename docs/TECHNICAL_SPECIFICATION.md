# Техническое задание: Corteris Tender AI

## 1. Цель
Windows-приложение для локального импорта, структурирования и анализа тендерной документации по системам безопасности, расчёта экономики и подготовки проекта пакета заявки. Юридически значимые действия остаются за пользователем.

## 2. Границы MVP
Работают: создание карточки, импорт ZIP/PDF/DOCX/XLSX/TXT/CSV, локальное извлечение текста, классификация документов, правиловой анализ юридических и конкурентных рисков, извлечение части оборудования, смета, скоринг, DOCX/XLSX документы. OCR, API площадок, серверная многопользовательская версия, ЭП и автоматическая подача — следующие этапы.

## 3. Архитектура
UI PySide6 → сервисы приложения → репозитории → SQLAlchemy → SQLite/PostgreSQL. Парсеры и аналитические движки подключаются через отдельные модули. Секреты хранятся через Windows Credential Manager/keyring.

## 4. Основные сценарии
1. Пользователь создаёт тендер. 2. Импортирует архив. 3. Система распаковывает и индексирует документы. 4. Пользователь запускает анализ. 5. Система выдаёт балл, риски и рекомендацию. 6. Пользователь корректирует смету. 7. Генерируется КП, таблица соответствия и запрос на разъяснение.

## 5. Интеграции
Официальные API/выгрузки ЕИС и ЭТП, ручной импорт, SMTP/IMAP, Telegram Bot API, OpenAI-совместимые API, Ollama. Каждый источник реализуется отдельным Connector с методами search(), fetch_metadata(), download_documents().

## 6. Безопасность
RBAC, аудит действий, шифрование секретов, локальный режим, проверка архивов, ограничение размеров, очистка временных файлов, резервное копирование БД, запрет автоматической подписи/подачи без подтверждения.

## 7. Тестирование
Unit: смета, скоринг, правила, парсеры. Integration: импорт архива и БД. UI smoke: создание/импорт/анализ/генерация. Security: traversal в ZIP, вредоносные имена, секреты, права ролей. Acceptance: демонстрационный тендер полностью проходит поток.

## 8. Развёртывание
Windows 10/11 x64; PyInstaller onedir; установщик Inno Setup; база и логи в профиле пользователя; обновления через подписанный manifest в последующей версии.
