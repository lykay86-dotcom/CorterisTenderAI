# Подключение OpenAI API

API-ключ нельзя создать автоматически из приложения ChatGPT или передать от имени пользователя.

ChatGPT Plus и OpenAI API имеют раздельный биллинг. Для подключения:
1. Войти на platform.openai.com под своей учетной записью.
2. Создать API Project.
3. Настроить платежный способ и лимит расходов.
4. Создать Secret API Key.
5. Скопировать ключ один раз.
6. Ввести ключ в Corteris Tender AI в разделе «Настройки интеграций».
7. Хранить ключ в Windows Credential Manager; не помещать его в документы, исходный код или .env, передаваемый другим лицам.

Рекомендуемые ограничения:
- месячный бюджет;
- предупреждение при 70% и 90% лимита;
- запрет отправки конфиденциальных документов без подтверждения;
- локальный режим Ollama как альтернативный вариант.
