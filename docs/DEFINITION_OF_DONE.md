# Definition of Done

Пункт RM завершён, когда:
- [ ] проведён аудит существующей реализации;
- [ ] отсутствует дублирующая бизнес-логика;
- [ ] работа выполнена в отдельной ветке с номером RM;
- [ ] бизнес-логика не находится в UI;
- [ ] деньги используют Decimal;
- [ ] даты содержат timezone;
- [ ] отсутствующие данные не выдумываются;
- [ ] критический стоп-фактор имеет приоритет;
- [ ] секреты отсутствуют в репозитории, БД и логах;
- [ ] изменение БД сопровождается миграцией;
- [ ] добавлены unit/integration/UI тесты;
- [ ] целевые тесты и полный pytest проходят;
- [ ] ROADMAP.md, STATUS.md и ROADMAP_HISTORY.md обновлены;
- [ ] PR объединён с main.

## Для AI
- [ ] AI не заменяет детерминированную логику;
- [ ] AI не изменяет подтверждённое решение;
- [ ] каждый вывод имеет источник;
- [ ] работает fallback без API;
- [ ] результат валидируется на добавление отсутствующих фактов.

## Для безопасности
- [ ] приватные ключи отсутствуют в клиенте;
- [ ] entitlement проверяется в сервисе, не только в UI;
- [ ] лицензия подписана;
- [ ] EXE, установщик и обновления подписаны;
- [ ] проверены подмена лицензии, времени, файлов и ответа сервера.
