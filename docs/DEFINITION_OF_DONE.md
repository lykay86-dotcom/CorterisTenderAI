# Definition of Done

Пункт дорожной карты считается завершённым, когда:

- [ ] определены критерии приёмки;
- [ ] существующий код проверен на дублирование;
- [ ] реализация находится в отдельной ветке;
- [ ] добавлены или обновлены тесты;
- [ ] весь набор тестов проходит;
- [ ] ошибки не скрываются общим `except`;
- [ ] интерфейс проверен вручную;
- [ ] база совместима со старыми данными;
- [ ] миграция добавлена, если изменилась схема;
- [ ] секреты не записываются в репозиторий;
- [ ] документация обновлена;
- [ ] создан Pull Request;
- [ ] изменения объединены с `main`;
- [ ] `ROADMAP.md` и `STATUS.md` обновлены;
- [ ] создана контрольная сборка, если менялся пользовательский сценарий.

## Обязательные проверки для RM-107

- [ ] подходящий тендер без рисков;
- [ ] высокий балл, но отсутствует обязательная лицензия;
- [ ] недостаточно документов;
- [ ] отрицательная маржинальность;
- [ ] высокая маржа, но неприемлемый срок;
- [ ] неподтверждённые данные;
- [ ] повторный расчёт после добавления документов;
- [ ] сохранение и повторная загрузка результата;
- [ ] совместимость со старой базой;
- [ ] проверка интерфейса.
