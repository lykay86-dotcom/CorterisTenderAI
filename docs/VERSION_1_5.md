# Corteris Tender AI 1.5

## Новый модуль «Мониторинг цен»

Версия 1.5 сравнивает предложения поставщиков только после проверки обязательных требований ТЗ. Самая низкая цена определяется по полной закупочной стоимости, а не по цене карточки товара.

Учитываются:
- количество и минимальная партия;
- скидка поставщика;
- включённый или невключённый НДС;
- доставка;
- наличие и срок поставки;
- гарантия;
- сертификат;
- официальность поставки;
- обязательный бренд и модель;
- допустимость эквивалента;
- технические характеристики.

Несоответствующее предложение нельзя автоматически добавить в смету.

## Источники

Поддерживается ручной импорт CSV, XLSX, JSON и XML. Для официальных API поставщиков предусмотрено последующее подключение через отдельные адаптеры. Автоматизация не должна обходить авторизацию, CAPTCHA или ограничения сайтов.

## Формула

Полная закупочная стоимость без НДС = товары после скидки + доставка + стоимость минимальной партии.

После выбора предложение передаётся в смету с эффективной стоимостью единицы, включающей распределённую доставку и минимальную партию.
