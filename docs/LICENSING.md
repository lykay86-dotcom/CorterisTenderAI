# Лицензирование и подписки

## Тарифы
Trial, Basic, Professional, Corporate, Custom.

## Entitlements
`tender_search`, `multi_provider_search`, `ai_summary`, `contract_analysis`, `counterparty_analysis`, `advanced_dashboard`, `export_reports`, `team_features`.

## Правила
- оплата проводится на сайте;
- карта не обрабатывается приложением;
- webhook идемпотентен;
- приложение получает подписанный токен;
- при неоплате платные функции блокируются, данные не удаляются;
- просмотр и экспорт собственных данных сохраняются;
- offline grace period ограничен.
