# Импорт из ЕИС

Версия 1.4 принимает:

- номер закупки длиной 10–25 цифр;
- ссылку домена zakupki.gov.ru, содержащую `regNumber` или номер закупки.

После нормализации создаётся карточка ЕИС. Документация загружается ручным архивом либо через официальный API/открытые данные, адрес которых пользователь добавляет в настройках площадок.

Программа не обходит CAPTCHA, двухфакторную аутентификацию и ограничения доступа.
