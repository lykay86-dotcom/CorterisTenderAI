# UI Guide

## Главное меню
Главная, Тендеры, Анализ, Контрагенты, Договоры, Сметы, Документы, Проекты, Аналитика, AI-помощник, Настройки.

## Раздел «Тендеры»
Управляющие кнопки удаляются из верхней панели и переносятся во вкладки: Поиск, Результаты, Реестр, Избранное, Проверка, Источники, Настройки площадок, История запусков, Ошибки.

## Новый дизайн
- адаптивное боковое меню;
- KPI-карточки;
- интерактивные графики;
- настраиваемые таблицы;
- центр уведомлений;
- фоновые операции;
- тёмная/светлая темы;
- DPI и accessibility;
- loading/empty/warning/error состояния.
