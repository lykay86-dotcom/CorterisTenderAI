# Corteris Tender AI 1.5

# Corteris Tender AI — рабочий MVP

Проект реализует сквозной локальный сценарий: карточка тендера → импорт документов → извлечение текста → анализ рисков → смета → решение об участии → генерация КП/таблицы соответствия/запроса.

## Запуск разработчика
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
pip install -r requirements.txt
python -m app.main
```

## Проверка ядра
```bash
pip install pytest
pytest -q
python scripts/create_demo.py
```

## Важные ограничения
Это MVP, а не готовая промышленная система для автоматической подачи. Правиловой анализ не заменяет юриста. DOC/XLS старых форматов, OCR, RAR и 7Z требуют дополнительных системных компонентов/настройки. Подключения к ЕИС/ЭТП должны выполняться через официально разрешённые интерфейсы.

См. `docs/TECHNICAL_SPECIFICATION.md`, `docs/ARCHITECTURE.md`, `docs/BUILD_WINDOWS.md`.
