# Установка полного патча 1.2.1

1. Закройте приложение.
2. Сделайте копию текущей папки проекта.
3. Распакуйте патч в корень проекта с заменой файлов.
4. Запустите тесты:

```powershell
.\.venv\Scripts\python.exe -m pytest -q
```

5. Соберите EXE:

```powershell
.\scripts\build_exe.ps1
```

При первом запуске база обновится автоматически. Старую SQLite удалять не требуется.

В настройках появится раздел «Диагностика БД» с резервным копированием, восстановлением, оптимизацией и экспортом.
