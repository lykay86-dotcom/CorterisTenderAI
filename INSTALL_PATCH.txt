Corteris Tender AI 1.5.1 — Sprint 1.2.1 Patch

1. Сделайте резервную копию проекта Sprint 1.1.
2. Распакуйте содержимое патча в корень проекта с заменой файлов.
3. Удалите старый файл app\database\models.py, если он остался.
4. Запустите тесты:
   .\.venv\Scripts\python.exe -m pytest -q
5. Соберите EXE:
   .\scripts\build_exe.ps1

Внимание: новая схема использует UUID. Старые базы с integer ID автоматически не мигрируются.
